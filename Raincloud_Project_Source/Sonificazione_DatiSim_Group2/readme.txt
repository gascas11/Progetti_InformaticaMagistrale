*************
* Requisiti *
*************
 
  1. Pure Data: https://puredata.info/downloads
  2. Installazione dei seguenti pacchetti esterni Pure Data: moocow e mrpeach
  3. Python 3
  4. Ide consigliati: Spyder, Jupyter

**********************************
* Esecuzione della sonificazione *
**********************************

  1. Aprire il file Raincloud_Sonificazione_DatiSim_G2.pd
  2. Eseguire il file Raincloud_Sonificazione_DatiSim_G2.py

*******************************************************
* Registrazione della sonificazione tramite Pure Data *
*******************************************************

  1. Cliccare sul messaggio "open" della sezione "Scrittura su audio" per avviare la registrazione
  2. Cliccare sul messaggio "stop della sezione "Scrittura su audio" per stoppare la registrazione

****************************************************
* Note relative all'esecuzione dello script python *
****************************************************

  1. Potrebbe essere necessario attendere qualche secondo per l'apertura dell'immagine relativa al Raincloud sonificato.
  2. Tutte le immagini generate verranno salvate nella cartella "img" della directory corrente.
